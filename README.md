# task

## Clone the Repository:

```sh
git clone https://github.com/mahmodghnaj/task.git
cd task
```

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build

```
